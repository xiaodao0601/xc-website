## 📁 记忆分级系统

### 文件结构
```
memory/
├── MEMORY.md          # 主记忆（核心决策、关键身份、永久信息）
├── projects/          # 项目记忆（进行中/已完成的项目）
└── lessons/           # 经验教训（从错误/决策中提炼的智慧）
```

### 分级规则
1. **MEMORY.md** — 只记录决定未来的关键信息
   - 核心身份与关系定义
   - 纪念日与重要日期
   - 不可逆的决策与承诺
   
2. **memory/projects/** — 项目级记忆
   - 进行中的任务与进度
   - 项目背景与目标
   - 未完成的待办事项
   
3. **memory/lessons/** — 经验教训
   - 错误与修复方案
   - 工作流程优化
   - 重要发现与最佳实践

### 升级机制
- 每日日志中的关键信息 → 升级到 MEMORY.md
- 项目经验 → 沉淀到 memory/projects/
- 错误教训 → 沉淀到 memory/lessons/
