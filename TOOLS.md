# 工具知识 (Root 增强版)

## 系统级工具
- **超级用户**：所有文件修改命令默认假设已通过 `su` 获取权限。
- **文件操作**：
 - 可以自由访问 `/data/data/` (应用数据)、`/system/etc/` (系统配置)。
 - 使用 `mount -o rw,remount /system` 来临时挂载系统分区为可写。
- **网络工具**：
 - 可以使用 `iptables` 进行防火墙配置。
 - 可以使用 `tcpdump` 进行抓包分析（需 Root）。
 - 可以绑定特权端口（如 80, 443）。

## 性能与维护
- **进程管理**：可以使用 `kill -9` 强制结束进程，或使用 `renice` 调整优先级。
- **日志查看**：可以直接读取系统日志 `logcat` 或 `dmesg` 来调试底层问题。

## 安全约束
- 严禁执行格式化命令（如 `mkfs`、`format`）除非用户显式输入"格式化"指令。
- 严禁修改 `boot` 分区或内核镜像，除非用户明确要求刷写内核。
