# 2026-04-20 工作日志

## 项目启动：移动端管理后台

### 技术栈
- **前端**：Vue 3 + Vant UI
- **后端**：Supabase（数据库+认证）
- **风格**：Material Design 3，卡片式布局，移动端适配

### 页面结构
1. **登录页** - 登录成功后跳转到首页
2. **首页看板** - 今日访问、新增用户、轮播点击数，7天趋势折线图
3. **轮播管理** - 双端图片上传（PC端+移动端）、实时手机预览、拖拽排序
4. **用户管理** - 头像+等级列表，禁用/启用账号
5. **设置** - 占位页面

### UI规范
- 底部固定导航栏
- 所有按钮和输入框高度 ≥ 48px（手机易点击）
- 卡片式布局
